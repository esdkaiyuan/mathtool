使用说明

有任何宝贵的意见，以及问题需要反馈，请前往https://www.esdkaiyuan.online/ 基本使用说明： 1. 左侧输入 LaTeX，MathJax 会实时渲染 SVG 矢量预览。 2. 符号库按钮只显示数学符号，点击后把对应 LaTeX 示例插入光标处。 3. 顶部字体下拉框会同步预览区域样式，并用于 Python 后端生成 Word 原生公式。 4. 点击“生成 Word”会调用本地 Python 后端，下载包含 Word 原生矢量公式对象的 DOCX。 5. Ctrl+Enter 渲染右侧公式，Ctrl+Shift+C 复制左侧 LaTeX，Ctrl+Shift+W 生成 Word。
